# feechecker

Get the latest btc block fee rates using node.js and blockchain.info

## Getting Started

clone the repo
```
git clone https://github.com/cryptom3rchant/feechecker.git
```

install the dependencies
```
npm install
```

run the program

```
node index.js
```

## Result format 

{ fee satoshis per byte: number of transactions, ... }

```
{
  "5": 27,
  ...
}
```
